Portfolio

The Pig Game

A simple jeopardy dice game in which two players race to reach 100 points. Each turn, a player repeatedly rolls a die until either a 1 is rolled or the player holds and scores the sum of the rolls (i.e. the turn total).

The Fruit Shop

A fruit shop that lists fruits and allows users to create a shopping basket. The user can add and remove items from the basket, and can see the contents of their basket on the page.

Sega Tribute Page

A tribute page to celebrate the Classic Sega game "Streets of Rage". The page was built with HTML, styled with CSS and uses a bit of JavaScript to display the current date.

Comment Box

A form that allows users to enter their name, email and a short comment and submit it. When they submit the form, the page then shows thier comment in a section alongside the author's name.

Theme Clock

An analogue and digital clock that tells you that time as well as the date. This was also an opportunity for me to practice how to change the themeing and therefore the clock has light and dark theme.
